# Project Title: Helpdesk Chat - Multi-Platform AI Assistant
This project implements a responsive chat application based on the provided technical assessment mockup using the Flutter framework. It features multi-platform compatibility (Desktop, Tablet, Phone) and integrates a large language model (LLM) to act as an automated helpdesk agent
# Setup and Configuration
1. Prerequisites
    * Flutter SDK (Version compatible with google_generative_ai: 0.4.7)
    * Dart SDK
    * VS Code or Android Studio
2. Install Dependencies
Ensure your `pubspec.yaml` is configured, then run:
```
flutter pub get
```
